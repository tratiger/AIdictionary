# AIdictionary
Notionにわからない単語を書いておくだけで自動で意味と例文と語源をNotionに保存する（一時間おきに自動更新）

ChatGPTのAPIとNotionAPIを取得して貼り付けるだけで動く 

実行環境はGAS

Google App Script GitHub アシスタントでGASをgithubで管理できる
